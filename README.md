## Projects

### Project 4 — NYC Taxi Jan 2023: End-to-End Data Pipeline
**Tools:** Python, SQLite, Power BI | **Dataset:** NYC TLC Trip Records | **Scale:** 3M+ rows

**Pipeline:** Raw Parquet → `load_data.py` → SQLite → `clean_data.py` → `analysis.py` → Power BI Dashboard

**Key Findings:**
- Peak hour is 6pm with 214,250 trips — highest demand of the day
- Tuesday is the busiest day generating $13.4M revenue — higher than Friday or Saturday
- 4am has the highest average fare at $32.09 — airport runs dominate late night
- Sunday earns more per trip ($28.49) than Saturday ($25.60) — longer routes

**Recommendation:** Deploy maximum fleet at 6pm on weekdays and maintain late-night availability for high-fare airport routes.
