# Project 3: HR Analytics Dashboard — Power BI

## Overview
An interactive Power BI dashboard analyzing 1,470 employee 
records from the IBM HR Analytics dataset. Designed to help 
HR managers understand attrition patterns, salary distribution 
and workforce demographics through visual KPIs and interactive 
charts.

## Dashboard Features
- 📊 KPI Card — Overall Attrition Rate (16.12%)
- 🏢 Department Wise Attrition — bar chart showing which 
  department loses the most employees
- 💰 Salary by Job Role — horizontal bar chart showing 
  compensation across all job roles
- 👥 Age Distribution — histogram showing workforce 
  age spread
- 🔍 Interactive Slicers — filter by Department, 
  Job Role, Gender and Age Group

## Business Questions Answered
1. What is the overall employee attrition rate?
2. Which department has the highest attrition?
3. How does salary vary across job roles?
4. What is the age distribution of the workforce?
5. Which employee segments are most at risk of leaving?

## Key Insights
- 🚨 Attrition Rate: 16.12% — above 15% industry average
- 🔴 R&D department has the highest employee attrition
- 💰 Managers earn the most — Sales Reps earn the least
- 📊 20x salary gap — $1,009 minimum to $19,999 maximum
- 👥 Average employee age: 37 years
- ⚠️ Low compensation likely driving high attrition

## Tools Used
- Power BI Desktop
- Dataset: Kaggle IBM HR Analytics (1,470 employees, 35 columns)
- Data previously analyzed in Project 2 using Python

## How to View
1. Download the .pbix file from this repository
2. Open with Power BI Desktop (free download)
3. Use slicers to filter by Department, Job Role or Age Group
4. All visuals are interactive — click any chart to 
   cross-filter the entire dashboard
